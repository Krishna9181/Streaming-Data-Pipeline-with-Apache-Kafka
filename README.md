# Streaming-Data-Pipeline-with-Apache-Kafka
* Developed a real-time streaming pipeline using Kafka to ingest toll traffic data, process continuous streams, and load into a database for live analytics.
* Implemented data producers/consumers for high-throughput streaming; handled fault-tolerant processing, schema evolution, and sink integration with database visualization of results.
* Apache Kafka, Python, SQL, Streaming, Real-time Data Processing
